Our goal is simple

Frontendevents.com has just one goal, list all active conferences/meetups dedicated on the front-end development. Make a list is long and pretty hard so feel free to contribute.

If you want to contribute to the project by adding a conference/meetup not yet listed you can submit it via our github and post an issue with the event's infos.
